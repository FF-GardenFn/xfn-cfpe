# ANALYZER

Sentiment classifier and scraper pipeline for Claude product feedback. Designed to fetch customer sentiment from public sources, classify it using Claude, and categorize feedback by product area and sentiment polarity. Intended to surface actionable insights about Claude's perceived strengths and pain points.
